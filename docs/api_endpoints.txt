# API ENDPOINTS

## Аутентификация
POST   /auth/register      - Регистрация нового пользователя
POST   /auth/login         - Вход (получение токена)
POST   /auth/refresh       - Обновление токена

## Пользователь
GET    /user/profile       - Получить профиль
PUT    /user/profile       - Обновить профиль
GET    /user/personality   - Получить психотип
POST   /user/personality   - Отправить ответы на вопросы (определить психотип)

## Привычки
GET    /habits             - Список всех привычек
POST   /habits             - Создать новую привычку
GET    /habits/{id}        - Детали одной привычки
PUT    /habits/{id}        - Редактировать привычку
DELETE /habits/{id}        - Удалить привычку
POST   /habits/{id}/log    - Отметить выполнение

## Статистика
GET    /habits/{id}/stats  - Статистика по конкретной привычке
GET    /stats              - Общая статистика

## Аналитика
GET    /insights           - Получить инсайты от Джарвиса

## Контекст
POST   /context            - Отправить данные о местоположении/активности

## Чат
WS     /chat               - WebSocket соединение для чата с Джарвисом
